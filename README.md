<h1>TRADUTOR SIMPLES</H1>

### Projeto desenvolvido com: Javascript ES6, CSS 3 e HTML 5.

#### Esse tradutor permite que você traduza frases e palavras em:

* Português
* Espanhol
* Inglês
* Italiano
* Japonês
